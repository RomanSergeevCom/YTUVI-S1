# YTUVI — Season 1 Map

## Канал-«формула» для каждого выпуска

```
Hook → Geology → Свойства → Месторождение → Облагораживание / синтетика → Рынок / аукционы → Сертификат → Вынос
```

Эта формула — главный актив сезона: повторяемая структура даёт зрителю рамку и позволяет переиспользовать explainer-блоки.

## Повторяемые модули (reuse)

Чтобы не снимать одно и то же по 5 раз, делаем 3 **шаблонных шот-пака**, которые вшиваем во все выпуски сезона:

1. **GEM-FORMATION pack** — анимация 3 типов месторождений, 10–15 сек, используется в Geology-блоке.
2. **CERT-READ pack** — разворот типового GRS/SSEF-отчёта с подсветкой ключевых полей (Species / Variety / Origin / Treatment). Меняется только «шапка» под конкретный камень.
3. **LAB-MAP pack** — мировая карта лабораторий с пинами GRS (Lucerne/BKK/NY), SSEF, Gübelin, GIA. Повторно используется в T4.

## Production order (предложение)

1. **Пилот S1**: YTUVI-006 (Flagship explainer 5–8 мин) — snap-версия, минимальная логистика, one-shot сессия с Натальей. Даёт раннее видео на канале + тестирует темп/свет/язык/визуальный ключ.
2. **Шот-паки**: GEM-FORMATION + CERT-READ + LAB-MAP (можно без эксперта, just b-roll/motion) — переиспользуются во всех глубоких выпусках.
3. **Глубокий пилот**: YTUVI-001 (Рубин) — закладывает формулу, самый яркий insider-нарратив (Бирма, Harry Winston).
4. **Далее**: 004 (Шпинель+Корона — виральный «crown jewels» крючок + Бриллиантовая кладовая как опция), 005 (Изумруд — горячая тема облагораживания), 002 (Сапфир+Падпараджа), 003 (Гранаты).
